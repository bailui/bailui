## 重点方向

### 法律科技
- 维权与争议处理流程设计、文书生成体系、案件信息结构化整理
- 法律服务落地页、用户承接页面、咨询转化与业务流程组织

### 金融 / 加密
- 金融研究框架、加密市场观察、策略迭代与工具化呈现
- 指标面板、研究工作流、交易辅助信息系统与自动化分析流程

### 网站 / 产品 / 自动化
- 博客系统、导航平台、业务官网、内容矩阵与个人品牌站点
- 将运营、内容、工具、服务入口整合为一套可持续运转的系统
